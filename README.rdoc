== README

App developed for iCasei portal test.

Things you may want to cover:

* Ruby version
	ruby 2.1.5p273
* System dependencies
	MongoDB service must be installed, more info https://www.mongodb.org/
* Configuration
	<tt> $ bundle install</tt><br />
	Edit file /config/mongoid.yml section hosts with address service mongodb<br />
	<tt> $ rails server --binding localhost</tt>

* How to run the test suite
	<tt> $ rake -test </tt>
